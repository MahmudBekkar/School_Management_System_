# School Management System 🏫

A desktop application designed to manage school activities, student records, and educational data efficiently. This project was developed as part of the Object-Oriented Programming (OOP) course.

## 🚀 Features
* **Full CRUD Operations:** Create, Read, Update, and Delete student records.
* **User Authentication:** Secure login system for administrators/teachers.
* **Modern UI:** Clean and responsive user interface designed using specialized UI frameworks.
* **Database Integration:** Reliable data storage managed through SQL.

## 🛠️ Built With
* **Language:** C#
* **Framework:** .NET Framework (Windows Forms)
* **Database:** Microsoft SQL Server / PostgreSQL
* **UI Customization:** Guna UI2 / DevExpress

## 📂 Project Structure
The repository includes clean source code, filtered to exclude unnecessary compilation files (`bin/`, `obj/`, `.vs/`) for optimal performance and clean version control.
